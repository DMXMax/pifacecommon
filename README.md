pifacecommon
============

Common functions for interacting with PiFace products. Also available from 
[PyPI](https://pypi.python.org/pypi/pifacecommon/)

Installation
=================

    $ git clone https://github.com/piface/pifacecommon.git
    $ cd pifacecommon
    $ sudo python3 setup.py install
